# Vue Three Editor

# Information:
  Work In Progress!

  To develop vuejs threejs editor in module design format.


  troisjs package can render client only and not SSR server render.

# Set up:
  Need to install nodejs.


```
npm install
npm run ssrbuild
npm run generate
npm run dev
```
